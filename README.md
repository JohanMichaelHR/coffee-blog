# ☕ Coffee Blog

A fully responsive coffee blog website built with HTML5 and CSS3.  
This project focuses on modern layout techniques, clean semantic structure, and scalable CSS architecture.  
It showcases responsive design principles using Flexbox and CSS Grid to ensure an optimal user experience across different devices and screen sizes.
